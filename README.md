# GlobalTech-Sales-Analysis

📌 Overview

This project introduces data analysis using Pandas by working with a simulated dataset of GlobalTech sales. It focuses on exploring, filtering, and analyzing sales data to extract meaningful business insights.

The dataset represents quarterly sales transactions across multiple regions, product categories, and promotions.

⚙️ Technologies Used
Python
Pandas
NumPy
📁 Features
1. Data Loading & Exploration
Loads dataset from CSV format
Displays:
First 5 rows
Data types and structure
Dataset dimensions
Summary statistics
2. Basic Data Analysis
Calculates:
Total units sold
Total revenue
Average unit price
Selects key columns for quick analysis
3. Data Filtering
Filters data based on:
Region (e.g., North America)
High-volume sales (Units > 20)
Specific product promotions (e.g., PhoneX on promotion)
Time period (February 2024)
4. Advanced Analysis
Identifies:
Best-performing product
Sales by region (sorted)
Average units sold per category
Compares:
Promotional vs non-promotional sales performance
5. Missing Data Detection
Identifies columns with missing values
Calculates percentage of missing data per column
6. Business Insights
Determines:
Top-performing category in each region
Average price by category
Revenue contribution by each product
Calculates percentage share of total revenue
7. Sales Report

Generates a summary report including:

Total revenue
Total units sold
Average sale value
Regional performance breakdown
▶️ How to Run
Install required libraries:
pip install pandas numpy
Run the script:
python module09-assignment.py
📌 Notes
Dataset is simulated for learning purposes
Includes intentional missing values for analysis practice
Demonstrates foundational Pandas operations used in real-world data analysis
